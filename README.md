# Final Project: Building and Serving a Machine Learning Model with Streamlit

Фінальний проект курсу AI Fundamentals.

Проект складається з двох частин:

1. Тренування простої лінійної регресії на синтетичному датасеті (model.py).
2. Вебзастосунок на Streamlit, який робить прогноз за введеним значенням ознаки та показує на графіку різницю між прогнозом і фактичним значенням (app.py).

## Структура проекту

- model.py - генерація датасету, тренування моделі, оцінка якості, збереження моделі та даних
- app.py - Streamlit застосунок для прогнозування та візуалізації
- requirements.txt - список залежностей

Після запуску model.py створюються файли:

- linear_regression_model.joblib - натренована модель
- X.joblib, y.joblib - збережений датасет
- metrics.joblib - метрики якості моделі для сайдбару застосунку

## Як запустити

```bash
# 1. Встановити залежності
pip install -r requirements.txt

# 2. Натренувати і зберегти модель
python model.py

# 3. Запустити вебзастосунок
streamlit run app.py
```

Застосунок відкриється в браузері за адресою http://localhost:8501

## Що реалізовано за завданням

Частина 1 (model.py):

- train_regression_model - тренує модель LinearRegression на тренувальних даних
- save_regression_model - зберігає модель у файл linear_regression_model.joblib за допомогою joblib.dump
- evaluate_regression_model - робить прогноз на тестовій вибірці та виводить у термінал Mean Squared Error
- save_initial_datasets - зберігає датасет у файли X.joblib та y.joblib

Частина 2 (app.py):

- load_and_predict - завантажує модель з файлу та повертає прогноз для введених даних
- create_streamlit_app - будує інтерфейс: заголовок, слайдер для вибору значення ознаки в діапазоні від -3.0 до 3.0, кнопка "Predict value" та вивід прогнозу
- visualize_difference - будує графік: весь датасет сірими точками, фактичне значення синьою точкою, прогноз червоною точкою, пунктирна лінія між ними та підпис різниці

## Перевірка результату

Для значення ознаки 1.27 застосунок видає прогноз 26.509899021941507 і різницю -37.33. Це збігається з прикладом зі скриншота в умові завдання.

Метрики моделі на тестовій вибірці:

- MSE: 416.81
- RMSE: 20.42
- MAE: 16.83
- R2: 0.80

## Додаткові можливості (поза базовим завданням)

Ідеї для доповнень взяті з практичних завдань курсу в Google Colab.

1. Розширені метрики якості в model.py. Крім обов'язкового MSE виводяться RMSE, MAE та R2, так само як у задачі House Price Prediction з блоку Machine Learning. Метрики зберігаються у файл metrics.joblib.
2. Сайдбар "About the model" у застосунку. Показує рівняння натренованої моделі (коефіцієнт і зсув) та метрики з тестової вибірки. Якщо модель ще не натренована, застосунок підказує спочатку запустити python model.py.
3. Лінія регресії на графіку. Вмикається чекбоксом у сайдбарі. Показує поведінку моделі на всьому діапазоні ознаки: прогноз завжди лежить на цій лінії.
4. Панель метрик над графіком. Три картки: фактичне значення, прогноз і різниця. Дає змогу порівняти числа одразу, без вчитування у графік.
